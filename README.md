## Swisstronik-code challenge-#3

## Usage

### Clone repo

Make sure already installed nodejs and npm

```bash
git clone https://github.com/genznodes-ecosystem/code-challenge-3.git
cd code-challenge-3
npm i
```

### Run

Change RPC_URL and priv_key in `hardhat.config.js` 

- Compile

```bash
npm run compile
```

- Deploy

```bash
npm run deploy
```
